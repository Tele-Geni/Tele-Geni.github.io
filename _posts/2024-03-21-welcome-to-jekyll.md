这是我的第一篇博客文章。我使用 Jekyll 和 GitHub Pages 搭建了这个博客。

## 为什么选择 Jekyll？

- 简单易用
- 支持 Markdown
- 与 GitHub Pages 完美集成
- 丰富的主题和插件
- 完全免费

## 如何开始写作？

1. 在 `_posts` 目录下创建新的 Markdown 文件
2. 文件名格式为：`YYYY-MM-DD-title.md`
3. 在文件开头添加 YAML 头信息
4. 使用 Markdown 语法编写内容
